Содержание программы : 
4 класса ( Logger-работа с файлами;Service1-описание событий OnStop() и OnStart();Program,Installer1-установка службы)
Класс Options отвечает за настройку службы
XMLFile.xml - файл класса Options в формате  xml
XMLFile.xsd- описание схемы валидации XMLFile.xml
JsonFile.json - файл класса Options в формате  Json
ArchiveManager- описание процесса архивации и распаковки
EncryptManager- описание процесса шифрования и дешифрования
ParserXML & ParserJson- описание парсеров xml & json
Exception.txt- обработка исключений
